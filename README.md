# Healthcare Executive Business Dashboard

## Overview
This repository showcases a Business Intelligence (BI) project focused on healthcare operations and financial performance. It includes a synthesized healthcare executive dataset and a final exported dashboard report used to track key performance indicators (KPIs) at the executive level.

## Project Assets

* **`Executive_dashboard.pdf`:** A static export of the final executive dashboard. This visualization highlights operational efficiency and financial health, providing a snapshot of how executive leadership might consume this data.
* **`Healthcare_Business_Executive_Dataset.csv`:** The underlying dataset used to generate the dashboard. 

## Key Metrics Analyzed
The dataset contains critical healthcare business metrics, including:
* **Operational Metrics:** Length of Stay (LOS), Bed Days, Readmission status, and Encounter volumes across different departments.
* **Financial Metrics:** Treatment Revenue, Treatment Costs, and overall Margins.
* **Patient Demographics:** Age, Gender, Condition, and Payer Type (Insurance).

## How to Use
1. **View the Dashboard:** Open the `Executive_dashboard.pdf` to view the final reporting structure and visual analytics.
2. **Explore the Data:** The CSV dataset can be imported into BI tools like Tableau, Power BI, Looker, or Python/R for further exploratory data analysis (EDA) and custom visualization building.

> **Disclaimer:** The data provided in this repository is synthesized for educational and portfolio demonstration purposes. It does not contain any real Protected Health Information (PHI).
